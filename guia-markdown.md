# 📘 Guia Completo de Markdown

### 🐧 Uso Acadêmico e no Linux

---

## 🧠 Introdução

O **Markdown** é uma linguagem de marcação leve criada para facilitar a escrita e leitura de textos formatados.

Muito utilizada em:

* GitHub
* Documentação técnica
* Projetos Open Source
* Ambientes Linux

---

## ✍️ 1. Títulos

Use `#` para criar títulos:

```markdown
# Título 1
## Título 2
### Título 3
#### Título 4
```

---

## 🎨 2. Formatação de Texto

```markdown
**Negrito**
*Itálico*
~~Riscado~~
```

Resultado:

**Negrito**
*Itálico*
~~Riscado~~

---

## 📋 3. Listas

### Lista não ordenada

```markdown
- Item 1
- Item 2
  - Subitem
```

### Lista ordenada

```markdown
1. Primeiro
2. Segundo
3. Terceiro
```

## 🐧 10. Markdown no Linux

Markdown é amplamente utilizado no ambiente Linux para:

* 📄 Documentação de sistemas
* ⚙️ Scripts e organização
* 📚 Anotações técnicas

### Ferramentas comuns:

```bash
nano arquivo.md
vim arquivo.md
cat arquivo.md
less arquivo.md
```

---

## 🚀 11. Boas Práticas

* Use títulos para organizar o conteúdo
* Evite textos muito longos sem separação
* Utilize listas para facilitar leitura
* Padronize seus arquivos

---

## 🎓 Considerações Finais

O Markdown é uma ferramenta essencial para qualquer estudante ou profissional de tecnologia.

Dominar essa linguagem significa:

* Melhor organização
* Documentação mais clara
* Maior produtividade
