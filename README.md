## ⚙️ O que é Markdown?

Markdown é uma linguagem de marcação leve que permite formatar texto de forma simples e rápida.

Exemplo:

```markdown
# Título
## Subtítulo
**Negrito**
*Itálico*
```

Resultado:

# Título

## Subtítulo

**Negrito**
*Itálico*

## 🧠 Principais Sintaxes

### 📌 Títulos

```markdown
# H1
## H2
### H3
```

---

### 📌 Ênfase

```markdown
**Negrito**
*Itálico*
~~Riscado~~
```

---

### 📌 Listas

```markdown
- Item 1
- Item 2
  - Subitem
```

---

### 📌 Links

```markdown
[GitHub](https://github.com)
```

---

### 📌 Imagens

```markdown
![Texto alternativo](imagem.png)
```

---

### 📌 Código

```markdown
\`\`\`javascript
console.log("Hello World");
\`\`\`
```

---

### 📌 Citações

```markdown
> Isso é uma citação
```
